# movie_test_app

Flutter project for consume Movie API.

## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/ArdhanaGusti/movie_test_app.git
   ```
2. Open the project
   ```sh
   cd movie_test_app
   ```
3. Open terminal and run
   ```sh
   flutter pub get
   ```
4. Debug on emulator
   ```sh
   flutter run
   ```
